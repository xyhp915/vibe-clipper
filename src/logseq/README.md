# Logseq Plugin Source

This folder contains the source code for the Vibe Clipper Logseq plugin.
The plugin integrates with Logseq to provide clipping and note-taking functionalities,
enhancing productivity within the Logseq ecosystem.

## Key Components

- Plugin entry point and configuration files.
- Scripts for handling user interactions and data processing.
- Assets and utilities specific to Logseq's API.

For more details on the overall project, refer to the root README.md.
